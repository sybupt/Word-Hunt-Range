## ✨ 特性

- **三种经典游戏模式**：英译中、中译英、听音选词。听音模式自动朗读单词，训练听力。
- **场景选词模式**：加载自定义场景图片（如卧室、教室），单词卡片随机出现在图片中的标注位置，增强沉浸感。
- **系统原生语音**：在 Windows 上使用 SAPI 或 PowerShell 后备方案，macOS 使用 `say`，Linux 使用 `espeak`/`festival`，无需额外 API。
- **鼠标/体感枪控制**：支持鼠标点击选择答案，也可以与 AirGun 蓝牙体感枪配合，用光标瞄准并射击单词。
- **动态卡片布局**：每轮随机生成 5 个选项，卡片位置自动避让重叠，支持长文本智能截断。
- **实时反馈与得分**：选对绿色闪烁 +10 分，选错红色闪烁，自动进入下一题。
- **可调灵敏度与窗口**：鼠标灵敏度（10~50）、窗口尺寸（800x600 / 1024x768 / 1280x720）、全屏切换。
- **暂停与设置菜单**：游戏内按 ESC 可暂停，调整灵敏度、窗口大小、全屏后继续或退出。

## 🛠️ 安装与依赖

### 环境要求

- Python 3.7 或更高版本
- 操作系统：Windows / macOS / Linux（语音功能在各自平台原生支持）

### 安装依赖

<p>使用conda创建虚拟环境：</p>

```bash
conda create -n word_hunt python=3.9 -y
conda activate word_hunt
```
<p>切换到创建的conda环境后，直接pip下载依赖项即可</p>

```bash
pip install requirements.txt
```


如果不安装 pywin32，程序会自动使用 PowerShell 后备方案。

### 从源码运行

将 `wordhunt.py` 和 `scene_game.py` 放在同一目录下，并创建 `resource/` 文件夹结构（参见下文配置说明）。

```bash
python -m wordhunt
python wordhunt.py
```

## 🎮 快速开始

1. **运行游戏**：执行 `python wordhunt.py`，进入主菜单。
2. **选择模式**：
   - **经典模式**：选择词库（如 CET-4、小学英语）、游戏模式（英译中/中译英/听音选词）、游戏时长（30~180秒）和鼠标灵敏度。
   - **场景选词**：选择一个预置场景（如 bathroom、classroom），单词卡片将出现在场景图片的标注位置。
3. **开始游戏**：点击“开始游戏”，使用鼠标或体感枪瞄准单词卡片并点击左键。
4. **得分与继续**：每答对一题得 10 分，错误不扣分，自动刷新题目。
5. **结束**：时间耗尽后显示最终得分，可选择“再来一局”或“返回主菜单”。

## 🔧 配置与词库

### 词库格式（JSON）

词库存放在 `resource/` 目录下，每个 `.json` 文件对应一个词库。支持两种格式：

**格式一（推荐，带中文和音标）**：
```json
{
  "name": "CET-4",
  "1": {
    "english": "abandon",
    "chinese_trans": "v. 遗弃；放弃",
    "spell": "[əˈbændən]"
  },
  "2": {
    "english": "absence",
    "chinese_trans": "n. 缺席；缺乏",
    "spell": "[ˈæbsəns]"
  }
}
```

**格式二（仅单词列表）**：
```json
["apple", "banana", "orange"]
```

如果词库条目不足 5 个，游戏会自动重复抽取以保证选项数量。

### 场景模式配置

场景文件位于 `resource/scene/`，图片位于 `resource/scene_img/`。场景 JSON 格式：

```json
{
  "image_path": "scene_img/bathroom.jpg",
  "annotations": {
    "soap": {
      "chinese": "肥皂",
      "pos": [[120, 300], [130, 310]]
    },
    "mirror": {
      "chinese": "镜子",
      "pos": [[400, 150], [410, 160]]
    }
  }
}
```

- `image_path`：图片文件名（相对于 `resource/scene_img/`）。
- `annotations`：每个单词对应一个对象，`chinese` 为中文释义，`pos` 为一个或多个坐标 `(x, y)`，游戏会随机选一个坐标放置卡片。

首次运行程序时，如果 `resource/` 目录不存在，会自动生成默认的“CET-4”和“小学英语”词库示例。场景文件需手动创建。

## 📖 玩法说明

### 经典模式

- **英译中**：屏幕中央显示英文单词，下方/周围卡片显示中文释义，选择正确的中文。
- **中译英**：显示中文，选择对应的英文单词。
- **听音选词**：系统朗读一个英文单词，屏幕中央显示“请听发音并选择”，从卡片中选择正确拼写。语音会自动播放，可重复点击“听音”按钮（未实现重复按钮，但每次刷新题目都会自动朗读一次）。

### 场景选词模式

- 背景是一张真实场景图片（如浴室、教室）。单词卡片以半透明圆角矩形显示在图片的标注位置附近。
- 屏幕顶部显示当前需要射击的英文单词（如 `soap`），玩家需点击标注为“肥皂”的卡片。
- 卡片位置由 JSON 中的 `pos` 坐标决定，系统会根据屏幕缩放比例自动映射。
- 该模式更适合与体感枪配合：瞄准图片中的物体（如肥皂），扣动扳机击中对应卡片。

## ⌨️ 操作指南

| 操作                     | 按键/动作                     |
| ------------------------ | ----------------------------- |
| 选择选项（射击）         | 鼠标左键 / 体感枪左键         |
| 暂停/恢复游戏（经典模式）| ESC                           |
| 退出场景模式             | ESC（直接返回主菜单）         |
| 滚动场景列表（场景选择） | 鼠标滚轮                      |
| 调整滑块/按钮            | 鼠标拖拽或点击                |

游戏内暂停菜单提供“继续”、“设置”、“结束本局”、“返回主菜单”选项。设置中可以实时调整鼠标灵敏度和窗口大小。

## 🔮 场景模式制作

1. 准备一张背景图片（推荐 1920x1080 或 1280x720），放入 `resource/scene_img/`。
2. 在 `resource/scene/` 中创建同名 JSON 文件（例如 `bedroom.json`）。
3. 标注单词：确定你想让玩家射击的单词（如 `pillow`），给出中文释义和至少一个坐标（坐标可使用图像编辑软件查看像素位置）。
4. 启动游戏，在主菜单中选择“场景选词”，即可看到新场景。
