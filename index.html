我们的产品

🖱️

AirMouse

基于 ESP32 的无线交互设备，采用陀螺仪传感器技术，提供精确的光标控制和自然的人机交互体验。

  • 基于 ESP32 微控制器
  • 6轴陀螺仪 + 加速度计
  • 蓝牙 LE 无线连接
  • 电池管理和低功耗设计
  • PlatformIO 开发框架
🐝

VocaBee

一个引人入胜的英语词汇学习平台，采用 Godot 4.6 引擎开发，提供多种互动游戏模式和丰富的学习内容。

  • Godot 4.6 现代游戏引擎
  • 多种游戏模式
  • 6大词汇集合
  • 15个实景场景
  • 智能布局算法

核心功能

🎮 AirMouse - 智能交互

AirMouse 是一个创新的无线鼠标设备，通过陀螺仪传感器捕捉手部运动，提供自然直观的光标控制体验。完美适配 VocaBee 游戏，让学习变得更加互动有趣。

  • 实时陀螺仪数据处理
  • 死区和平滑过滤算法
  • 按键交互控制
  • 电池电量实时监测
  • 蓝牙设备配对
🖱️
ESP32 + 陀螺仪
🐝
Godot 4.6 引擎

📚 VocaBee - 趣味学习

VocaBee 是一个完整的英语学习生态，整合了音频识别、场景识别、评分系统等多个功能模块，让用户在游戏中自然获取词汇知识。

  • 听音选词游戏模式
  • 场景探索互动学习
  • 6个专业词汇集合
  • 动态卡片布局系统
  • 评分和成就系统
  • 文本转语音发音
